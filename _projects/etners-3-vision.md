### 📌 프로젝트 개요
해외 대학교의 등록금 고지서는 언어와 양식이 매우 다양하여 담당자가 수작업으로 지원 한도를 파악하는 데 큰 어려움이 있었습니다. 이를 해결하기 위해 **GPT-Vision, OCR, 그리고 MCP(Model Context Protocol)**를 결합하여 글로벌 학자금 지원 프로세스를 전면 자동화(AX)한 멀티모달 AI 어시스턴트입니다.

* **기술 스택:** GPT-4o(Vision), Claude 3.5 Sonnet, OCR(Polygon 인식), Tavily API, MCP, FastAPI, Redis
* **주요 기능:** 다국어 문서 시각적 번역(Overlay), 외부 대학 메타데이터 검색, 내부 이력 기반 지원금 자동 산출

---

### 💡 단계별 기술적 고도화 (Technical Evolution)

#### 1. Multimodal 기반 프로세스 자동화 (PoC)
초기에는 **GPT-Vision API**를 활용하여 전 과정을 구축했습니다. 회사의 지원 정책과 등록금 고지서 이미지를 입력으로 하여 데이터 추출부터 최종 지원금 산출까지 수행하는 파이프라인을 만들었습니다. 이를 통해 복잡한 비정형 문서의 의미론적 해석 가능성을 검증했습니다.

#### 2. OCR Polygon 기반의 시각적 번역 레이어 (UX 개선)
LLM의 텍스트 번역 결과만으로는 담당자가 원본 문서의 어느 부분이 번역되었는지 직관적으로 확인하기 어렵다는 피드백을 수용했습니다.
* **Solution:** OCR 엔진을 통해 텍스트의 **Polygon(좌표)** 데이터를 먼저 추출했습니다.
* **Overlay:** 추출된 좌표값에 번역된 텍스트를 매핑하여 원본 이미지 위에 번역본을 덮어씌우는 기능을 구현했습니다. 이를 통해 심사 담당자가 원문과 번역문을 시각적으로 즉시 교차 검증할 수 있도록 UX를 혁신했습니다.



#### 3. Tavily & MCP 기반의 Agentic Workflow 완성
전 세계 각국에서 들어오는 다양한 학교 정보를 검증하고, 개인별 한도를 정확히 계산하기 위해 기능을 확장했습니다.
* **Tavily API 통합:** 생소한 해외 학교의 경우, 실시간 웹 검색을 통해 학교의 실재 여부와 위치 등 메타데이터를 수집하여 디스플레이 함으로써 심사 신뢰도를 높였습니다.
* **MCP(Model Context Protocol) 활용:** LLM이 사내 DB에 안전하게 접근하여 신청자의 **이전 지원 내역**을 직접 조회하게 했습니다. 이를 통해 남은 한도를 계산하고, 정책에 따른 최종 지원금을 자동 산출하는 완전한 에이전트 구조를 확립했습니다.

### 📈 기대 효과 및 성과
* **AX(AI Experience) 실현:** 단순 텍스트 인식을 넘어 다국어 문서의 시각적 가독성을 확보하여 업무 몰입도 향상
* **검증 프로세스 정교화:** 외부 검색(Tavily)과 내부 데이터(MCP)를 결합하여 수기 검토 시 발생할 수 있는 휴먼 에러 방지
* **비용 최적화:** Vision API와 OCR 엔진의 적절한 조합을 통해 처리 속도 개선 및 운영 비용 절감
