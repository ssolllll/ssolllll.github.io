### 📌 프로젝트 개요

사내 커뮤니케이션 도구 부재 문제를 해결하기 위해 **Spring Boot** 기반의 채팅 시스템 백엔드를 설계하고 구현한 프로젝트입니다. 이후 회사의 AI 기술 도입 로드맵에 따라 AI 프로젝트로 전환되었습니다.

* **기간:** 2024.10 ~ 2024.11 (약 2개월)
* **역할:** 백엔드 설계 및 구현
* **기술 스택:** Spring Boot, MySQL, REST API

### 👨‍💻 주요 구현 내용

#### 1. 핵심 기능 백엔드 구현
- 채팅방 목록 조회, 상세 메시지 조회 API 개발
- 첨부파일 업로드/다운로드 처리 기능 구현
- 비즈니스 초기 기획 단계부터 참여하여 채팅 기능의 구조 설계와 로직 검토에 기여

#### 2. 재활용 가능한 독립 모듈 설계
- 이후 서비스 전반에 재활용 가능한 구조로 설계
- 독립 모듈 형태로 완성하여 향후 다른 서비스와의 연동을 고려한 API 구조 채택

### 📈 성과

* 약 2개월간 집중 개발하여 채팅 시스템 기초 확립
* 독립 모듈 형태의 백엔드 구현 완료 — 서비스 전반에 재활용 가능한 구조 설계
* 실사용 기반의 커뮤니케이션 도구 초기 구조 기획 및 구현 주도
